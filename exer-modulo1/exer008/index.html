Principais Formatações

Negrito

Nesse texto, para colocar a frase ou uma parte em negrito basta colocar a tag b (não semântica), ou colocar a tag STRONG (semântica)

Itálico

Para colocar a frase ou palavra em italico basta colocar a tag i (não semântica), ou pode colocar usando a tag em (semântica)

Texto Marcado

Podemos estar marcando uma palavra com a tag mark

Texto Grande e Pequeno

Para criar letras Grande basta usar a tag big, mas e melhor formatar a parte que você quer deixar grande no CSS, e para criar palavras pequenas nos texto e so usar a tag small

Deletar Texto

Para deletar um texto basta usar a tag DEL

Inserir um Texto

Para inserir um texto basta usar a tag INS

Texto Subrescrito

Essa tag serve para esse tipo de coisa: x3+b-25

Texto Subscrito

Essa tag serva para esse tipo de coisa: H2O
